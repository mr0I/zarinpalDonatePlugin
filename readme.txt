افزونه حمايت مالي توسط وب سرويس سيستم پرداخت زرين پال

== Description ==

اکثر پلاگين هايي که براي حمايت مالي در سيستم زرين پال وجود دارند از نوع آسان پرداخت هستند.

در هنگام استفاده از وب سرويس اين امکان وجود داره که نام سايت و لگوي مورد نظر شما به عنوان پذيرنده در سايت زرين پال نمايش داده بشه ولي در صورت استفاده از آسان پرداخت نام صاحب حساب به عنوان پذيرنده نمايش داده خواهد شد.

در روش پرداخت به کمک وب سرويس شما ميتونيد به راحتي صحت پرداخت را درون سايت خود داشته باشيد و همچنين تمامي اطلاعات پرداخت کنندگان را درون بانک اطلاعاتي وب سايت خود ذخيره کنيد و به سادگي به آن ها دسترسي داشته باشيد.

براي همين منظور پلاگيني را براي وردپرس آماده کردم که به سادگي ميتونيد يک بخش حمايت مالي درون سايت خودتون ايجاد کنيد و به سادگي اون را مديريت کنيد.

امکانات پلاگين:

استفاده از شرت کد [ErimaZarinpalDonate] در هر برگه و نوشته اي از سايت براي نمايش فرم پرداخت.

داراي منوي تنظيمات و نمايش جزئيات پرداخت ها در بخش مديريت ورد پرس.

امکان سفارشي کردن استايل فرم پرداخت

امکان سفارشي کردن جملات تاييد يا عدم تاييد پرداخت

امکان انتخاب واحد پولي بر اساس تومان/ريال

استفاده از وب سرويس nosoap براي پرداخت زرين پال

نمايش متن خطاهاي احتمالي در زمان پرداخت بر اساس استاندارد زرين پال

 

برقراري امنيت:

تمامي ورودي هاي کاربر توسط تابع filter_input در php براي جلوگيري فيلتر شده اند.

تمامي مغير هاي $_SERVER براي دريافت آدرس صفحه توسط تابع htmlspecialchars قبل از استفاده فيلتر شده اند.

براي اجراي کوئري بر روي بانک اطالاعاتي از توابع پيش فرض وردپرس استفاده شده است.

ورودي هاي ارسالي از سمت بانک که درون آدرس قرار ميگيرند به صورت مستقيم درون کوئري هاي بانک اطلاعاتي استفاده نشده اند.
